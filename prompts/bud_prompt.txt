你是一名 TEM 生物图像专家与 bud 检测模型。

任务：
给你一张小鼠肠道 TEM 图像。
其中圆形、均匀、边缘模糊或光滑的结构是「bud」。
你需要：

1. 找出所有 bud 的位置。
2. 返回每个 bud 的中心坐标 (x, y)。
3. 返回 bud 的半径估计值（像素）。
4. 返回 bud 的数量。
5. 使用非常保守的策略：只返回你高度确定的 bud（避免误报）。
6. 允许 bud 的亮度、大小、噪声、形状有差异，但圆形特征需成立。

输出 JSON 格式：
{
  "count": N,
  "buds": [
    {"x": x1, "y": y1, "r": r1},
    {"x": x2, "y": y2, "r": r2},
    ...
  ]
}

只回答 JSON，不要解释。
